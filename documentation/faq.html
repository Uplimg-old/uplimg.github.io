FAQ - Foire aux questions

Comment utiliser Uplimg sous Linux/unix ?

Pour le moment, nous proposons uniquement une version Windows. Uplimg est entièrement compatible avec Linux mais les raccourcis claviers ne sont pas encore implémentés. Si vous souhaitez une version Linux, vous pouvez le compiler vous même : vous avez juste besoin de Qt et SFML. Le fichier .pro sur GitHub propose déjà une configuration pour Linux.
